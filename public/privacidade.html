Política de Privacidade

Foco GeriClass

Última atualização: 15 de junho de 2026

1. Sobre este app

O Foco GeriClass é um aplicativo de produtividade pessoal (tarefas, foco/pomodoro, hábitos, metas e agenda). Esta política explica quais dados coletamos, como são usados e quais são os seus direitos. Ao usar o app, você concorda com as práticas descritas aqui.

2. Dados que coletamos

  • Dados de conta: nome e e-mail, informados no cadastro/login (gerenciados pelo Supabase Auth).
  • Conteúdo que você cria: tarefas, hábitos, metas, blocos de agenda, sessões de foco e a "Revisão do dia" (humor e anotações). Esses dados ficam vinculados à sua conta.
  • Google Agenda (opcional): se você conectar sua conta Google, o app acessa os eventos da sua agenda apenas para exibi-los junto do seu planejamento, e cria/atualiza eventos correspondentes aos blocos que você criar no app. O token de acesso fica armazenado no seu dispositivo.
  • Comandos para a IA: ao usar o Assistente de IA, o texto que você digita ou dita é enviado a uma função no nosso servidor, que consulta o modelo Google Gemini para interpretar o pedido e sugerir ações (criar tarefas, blocos, etc.).
  • Armazenamento local: o app guarda preferências (tema, lembretes) e uma cópia em cache dos seus dados no armazenamento local do navegador, para funcionar mesmo offline.

Não coletamos dados de localização, não usamos cookies de rastreamento e não exibimos publicidade.

3. Como usamos os dados

Usamos seus dados exclusivamente para fornecer o app: salvar e sincronizar seu conteúdo entre dispositivos, mostrar seus relatórios, enviar lembretes e processar seus comandos de IA. Não vendemos seus dados nem os usamos para publicidade.

4. Compartilhamento e provedores

Não compartilhamos seus dados com terceiros, exceto com os provedores estritamente necessários para o funcionamento do app:

  • Supabase — autenticação e banco de dados (onde seu conteúdo é armazenado com segurança).
  • Google — API do Google Agenda (apenas se você conectar) e o modelo Gemini (quando você usa a IA).
  • GitHub Pages — hospedagem do site.

5. Uso de dados do Google

O uso e a transferência de informações recebidas das APIs do Google pelo Foco GeriClass seguem a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado. Os dados da sua Google Agenda são usados somente para fornecer os recursos de agenda dentro do app, não são transferidos a terceiros e não são usados para publicidade ou treinamento de modelos.

6. Armazenamento, segurança e retenção

Seus dados trafegam por conexão segura (HTTPS) e são isolados por conta no banco (cada usuário só acessa o que é seu). Mantemos seus dados enquanto sua conta existir. Você pode desconectar o Google a qualquer momento nas configurações do app (o que revoga o acesso) e sair da conta para limpar os dados locais do dispositivo.

7. Seus direitos (LGPD)

Você tem direito a acessar, corrigir, exportar ou excluir seus dados. Para solicitar a exclusão da sua conta e de todos os dados associados, ou para qualquer dúvida sobre privacidade, entre em contato pelo e-mail abaixo.

8. Crianças

O app não é destinado a menores de 13 anos e não coletamos intencionalmente dados de crianças.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. A data da última atualização é indicada no topo desta página.
